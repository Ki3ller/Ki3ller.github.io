文案避雷规则

满堂春工作室 GalGame 避雷规则（精简版）
女主不洁 / 非初非处
女主角（尤其是可攻略的主要角色）不得有明确的前男友、与他人发生关系的经历或精神上的白月光。

绿帽 / 出轨 / 送女
禁止男主角被绿、女主角与男配角暧昧、或主角主动将女主推给别人。

虐主 / 憋屈
禁止男主角长期受欺辱、被压制却无法反击，导致玩家感到憋屈。

漏女 / 收女不全
前期出场且有明显好感暗示的女配角，必须在后续给予交代或可攻略，不能无端消失或配给路人。

百合 / 腐向
禁止可攻略女主之间发展百合关系，更禁止男主角搞基或腐向内容。

烂尾 / 腰斩
游戏必须有一个完整的结局，不能虎头蛇尾或关键线路突然中断。

脑残反派 / 降智剧情
反派（情敌或阻碍者）的行为逻辑不能过于弱智，不能依靠对手强行降智来推进剧情。

三观不正
禁止宣扬反人类、反社会、极端自私自利等错误价值观。

剧情注水
日常剧情应有意义，不能大量重复无意义的对话或灌水延长游戏时间。

作者/制作组失信
如果之前作品有严重雷点或长期断更，新作会被自动列入避雷名单。

感谢您对健康 GalGame 环境的支持。
