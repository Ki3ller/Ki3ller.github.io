专升本数学（二 方程式的解法）

发表于2025-12-07|更新于2025-12-07

一元二次方程开根法完全指南

一、基本概念

1.1 什么是开根法？

开根法是解一元二次方程的一种方法，适用于方程能化为 “（某式）² = 常数” 的形式。

1.2 核心思想

如果一个数的平方等于 (a)，那么这个数就是 (a) 的平方根，记作 (\pm\sqrt{a})。

二、三种常见类型及解法

类型1：直接开平方 (x^2 = k)

步骤：

  1. 方程已是 (x^2 = k) 形式
  2. 两边开平方：(x = \pm\sqrt{k})
  3. 写出两个解

例题：
[
x^2 = 25 \quad\Rightarrow\quad x = \pm 5
]

类型2：完全平方形式 ((x \pm m)^2 = k)

步骤：

  1. 开平方：(x \pm m = \pm\sqrt{k})
  2. 分成两个一次方程：
    • (x \pm m = \sqrt{k})
    • (x \pm m = -\sqrt{k})
  3. 分别解出 (x)

例题：
[
(x-3)^2 = 16
]
[
x-3 = \pm 4
]
[
\begin{cases}
x-3 = 4 & \Rightarrow x = 7 \
x-3 = -4 & \Rightarrow x = -1
\end{cases}
]

类型3：需要先整理的形式 (ax^2 = c) 或 (ax^2 + b = c)

步骤：

  1. 移项：把常数项移到右边
  2. 系数化为1：两边除以 (x^2) 的系数
  3. 开平方求解

例题：
[
4x^2 = 100
]
[
x^2 = 25 \quad\Rightarrow\quad x = \pm 5
]

[
2x^2 - 18 = 0
]
[
2x^2 = 18
]
[
x^2 = 9 \quad\Rightarrow\quad x = \pm 3
]

三、关键技巧总结

3.1 移项规则

  • 从左边移到右边：变号（加变减，减变加）
  • 目的：让平方项单独在一边

3.2 系数化为1

当方程形如 (ax^2 = c) 时：
[
x^2 = \frac{c}{a}
]

3.3 开平方后的处理

从 ((x \pm m) = \pm n) 到解出 (x)：

  • 消去加法用减法：(x + m = n \Rightarrow x = n - m)
  • 消去减法用加法：(x - m = n \Rightarrow x = n + m)

四、易错点提醒

  1. 忘记正负号：开平方必须写 (\pm)
  2. 计算错误：特别是负数运算
    [
    x + 4 = -5 \Rightarrow x = -5 - 4 = -9
    ]
  3. 系数化为1时除错：
    [
    9x^2 = 144 \Rightarrow x^2 = 16 \ (\text{不是 } x^2 = 12)
    ]

五、综合例题

例题1：直接型

[
x^2 - 64 = 0
]
解：
[
x^2 = 64 \Rightarrow x = \pm 8
]

例题2：完全平方型

[
(x+1)^2 - 16 = 0
]
解：
[
(x+1)^2 = 16
]
[
x+1 = \pm 4
]
[
x = 3 \ \text{或} \ x = -5
]

例题3：需要配方型（进阶）

[
x^2 + 6x + 9 = 36
]
解：

  1. 认出左边是完全平方：((x+3)^2)
  2. ((x+3)^2 = 36)
  3. (x+3 = \pm 6)
  4. (x = 3 \ \text{或} \ x = -9)

六、开根法 vs 十字相乘法

特征 开根法 十字相乘法
适用形式 能化为 ((\text{式})^2=k) 能分解为两个一次式乘积
常见类型 (x^2=k), ((x\pm m)^2=k) (x^2+bx+c=0)
关键步骤 开平方，取正负 找两个数满足积=c、和=b
解的数量 总是两个（可能相同） 总是两个（可能相同）

七、练习自查清单

做完题目后问自己：

  • 是否写成”（某式）²=常数”的形式了？
  • 开平方时写 (\pm) 了吗？
  • 解一次方程时移项符号对吗？
  • 最后的解有两个吗（除非两解相同）？

八、学习心得

开根法的本质是降次——把二次方程降为一次方程来解。关键是要善于识别方程是否可以表示为平方形式，然后利用平方根的定义求解。

掌握了开根法，你就为学习更一般的配方法和公式法打下了坚实基础。
